Proposito del sitio

El proposito de este sitio es conocer cuales son las etiquetas basicas o mas utilizadas ya que esta es la estructura básica del HTML, y estas a su vez contienen otras propiedades, como son los atributos y su valor los cuales veremos en el desarrollo de este Objeto virtual de aprendizaje.

Etiquetas basicas HTML

Las etiquetas en Html son fragmento de codigos encerados por < > que nos permiten crear elementos. Acontinuacion veremos un listado de las etiquetas basicas o mas usadas en Html.

< header>...</header> Para definir la cabecera de la página.
<footer>...</footer> Para definir el pie de página.
<nav>...</nav>: Para definir un elemento de navegación, como el menú principal o secundarios.
<section>...</section> Se utilizan para encerrar el código correspondiente a una sección.
<aside>...</aside> delimita todo aquel contenido que no es directamente contenido principal.
<article>...</article> Se utiliza para definir artículos.
<head>: para información sobre el documento.
<b>...</b> Aplica negrita al texto.
<i>...</i> Aplica cursiva al texto.
<u> </u> Aplica subrayado.
<font>...</font> Delimita un texto con un formato de fuente.
<h1>...</h1> delimitaría el tipo de fuente de mayor tamaño.
<br> Introduce un salto de línea.
<p>... </p> Delimita un párrafo de texto.
<hr> Inserta una línea horizontal.
<div> Se utiliza para delimitar una sección dentro del documento.
<ul> </ul> listas no numeradas.
<ol></ol> listas numeradas.
<li></li> elemento de la lista.
<img src="dirección de la imagen"> Inserta una imagen que se encuentra en la ruta.
<table>…</table> Define dónde comienza y termina la tabla.
<tr>…</tr> Indica el comienzo y el fin de cada una de las filas de la tabla.
<td>...</td> Indica el comienzo y el fin de cada una de las columnas o celdas dentro de las filas.
<td>...</td> Indica el comienzo y el fin de cada una de las columnas o celdas dentro de las filas.

Atributos y valor

Teniendo en cuenta el gran número de etiquetas, estas no son suficiente para crear páginas complejas. Algunos elementos como las imágenes y los enlaces requieren cierta información adicional para estar completamente definidos. La etiqueta por ejemplo se emplea para incluir un enlace en una página. Utilizando sólo la etiqueta no es posible establecer la dirección a la que apunta cada enlace. Como no es viable crear una etiqueta por cada enlace diferente, la solución consiste en personalizar las etiquetas HTML mediante cierta información adicional llamada atributos. De esta forma, se utiliza la misma etiqueta para todos los enlaces de la página y se utilizan los atributos para indicar la dirección a la que apunta cada enlace. No todos los atributos se pueden utilizar en todas las etiquetas. Por ello, cada etiqueta define su propia lista de atributos disponibles. Además, cada atributo también indica el tipo de valor que se le puede asignar y este valor del atributo siempre se encierra con comillas. Si el valor de un atributo no es válido, el navegador ignora ese atributo. Aunque cada una de las etiquetas HTML define sus propios atributos, algunos de los atributos son comunes a muchas o casi todas las etiquetas. De esta forma, es habitual explicar por separado los atributos comunes de las etiquetas para no tener que volver a hacerlo cada vez que se explica una nueva etiqueta. Los atributos comunes se dividen en cuatro grupos según su funcionalidad:
1) Atributos básicos: se pueden utilizar prácticamente en todas las etiquetas HTML.
2) Atributos para internacionalización: los utilizan las páginas que muestran sus contenidos en varios idiomas o aquellas que quieren indicar de forma explícita el idioma de sus contenidos.
3) Atributos de eventos: sólo se utilizan en las páginas web dinámicas creadas con JavaScript.
4) Atributos para los elementos que pueden obtener el foco.
